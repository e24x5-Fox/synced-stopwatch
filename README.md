# Synced Stopwatch

Секундомер, который работает одинаково на нескольких устройствах в одной
Wi-Fi сети: нажал «Старт» на телефоне — он сам запустился и на компьютере,
и на другом телефоне рядом. Ничего настраивать не нужно — устройства
находят друг друга сами.

## Скачать

Готовые файлы — на странице
**[Releases](https://github.com/e24x5-Fox/synced-stopwatch/releases/latest)**.

| Платформа | Файл |
|---|---|
| Android | `SyncedStopwatch.apk` |
| Windows | `SyncedStopwatch.exe` |

Просто зайди по ссылке выше, открой самый последний релиз и скачай нужный
файл — устанавливать ничего дополнительно не нужно.

## Установка

### Android
1. Скачай `SyncedStopwatch.apk` на телефон.
2. Открой файл. Android покажет предупреждение про «установку из
   неизвестных источников» — это нормально для приложений не из Google
   Play, разреши установку.
3. После установки открой приложение как обычное.

### Windows
1. Скачай `SyncedStopwatch.exe`.
2. Запусти файл двойным кликом.
3. Windows Defender (SmartScreen) может показать предупреждение о
   неизвестном издателе — нажми **«Подробнее» → «Выполнить в любом
   случае»**.
4. При первом запуске Windows спросит разрешение на доступ к сети —
   обязательно разреши, иначе синхронизация с другими устройствами не
   заработает.

Ничего устанавливать сверх этого не нужно — оба файла самодостаточны.

## Как пользоваться

1. Открой приложение.
2. При желании впиши название вверху экрана — для чего ведёшь отсчёт
   (например, «Без сигарет», «С начала проекта»).
3. Нажми **Старт** — секундомер пошёл.
4. Открой то же приложение на другом устройстве в той же Wi-Fi сети —
   оно само подхватит текущее время и название в течение пары секунд.
5. **Стоп** и **Сбросить** тоже применяются сразу ко всем устройствам.
6. Внизу экрана строка «Устройств рядом: N» показывает, сколько ещё
   копий приложения сейчас видно в сети.

Секундомер сохраняет своё состояние на диске: если закрыть приложение,
выключить телефон или компьютер — при следующем открытии время досчитается
правильно, как будто ничего не выключалось.

## Требования

- Все устройства должны быть подключены к **одной и той же Wi-Fi сети**.
- Через мобильный интернет (LTE/5G) синхронизация не работает — нужен
  именно общий Wi-Fi.
- Некоторые роутеры (особенно в кафе, отелях, офисах) включают «изоляцию
  клиентов» — тогда устройства физически не видят друг друга в сети. Это
  настройка роутера, приложение тут ничего сделать не может.

## Если что-то не работает

- **Секундомеры не синхронизируются** — проверь, что оба устройства в
  одной Wi-Fi сети, и что на роутере не включена изоляция клиентов.
- **Windows блокирует запуск** — разреши доступ в брандмауэре Windows
  Defender при первом запуске.
- **Android не даёт установить APK** — разреши установку из неизвестных
  источников в настройках телефона (обычно всплывающее окно предлагает
  это сделать сразу).

Если проблема осталась — опиши её в
[Issues](https://github.com/e24x5-Fox/synced-stopwatch/issues) этого
репозитория.
