# Synced Stopwatch — сборка APK через GitHub Actions

## Что это
Секундомер на Kivy: один общий счётчик на все устройства в одной локальной
сети (Wi-Fi). Старт/стоп/сброс на любом телефоне применяется ко всем —
синхронизация через UDP-broadcast (порт 5005). Состояние также сохраняется
на диске, поэтому переживает перезапуск приложения.

## Шаги, чтобы получить APK

1. **Создай новый репозиторий на GitHub** (например, `synced-stopwatch`),
   публичный или приватный — не важно.

2. **Залей туда эти 3 файла/папки**, сохранив структуру:
   ```
   main.py
   buildozer.spec
   .github/workflows/build.yml
   ```
   Проще всего через веб-интерфейс GitHub: "Add file" → "Upload files",
   перетащить все файлы (не забудь про папку `.github/workflows/build.yml`
   — GitHub сам создаст нужные подпапки).

   Либо через git:
   ```bash
   cd synced_stopwatch
   git init
   git add .
   git commit -m "initial commit"
   git branch -M main
   git remote add origin https://github.com/ТВОЙ_ЛОГИН/synced-stopwatch.git
   git push -u origin main
   ```

3. **Зайди во вкладку "Actions"** в репозитории на GitHub.
   Сборка запустится автоматически после пуша (или нажми "Run workflow"
   вручную, если видишь такую кнопку — workflow_dispatch это разрешает).

4. Сборка идёт **10-25 минут** при первом запуске (Buildozer скачивает
   Android SDK/NDK с нуля). Следующие сборки будут быстрее за счёт кэша.

5. Когда сборка зелёная (успешна) — открой её, внизу будет раздел
   **Artifacts** → `synced-stopwatch-apk`. Скачай zip, внутри — файл `.apk`.

6. Перенеси `.apk` на телефон (через USB, облако, телеграм самому себе —
   как удобно) и установи. Android спросит разрешение на установку
   "из неизвестных источников" — это нормально для приложений не из
   Google Play.

## Как проверить синхронизацию
- Установи APK на 2+ телефона, подключи оба к **одному и тому же Wi-Fi**.
- Открой приложение на обоих.
- Нажми "Старт" на одном — через пару секунд (не более 3, из-за heartbeat)
  второй телефон подхватит и покажет то же время и статус "Идёт".

## Если синхронизация не работает
- Проверь, что оба устройства в одной Wi-Fi сети (не в гостевой, не в
  разных VLAN).
- Некоторые роутеры включают "изоляцию клиентов" (AP/Client isolation) —
  тогда устройства физически не видят broadcast друг друга. Это настройка
  роутера, лечится в его админке.
- Мобильный интернет (LTE/5G) для этого не подходит в принципе — нужен
  именно общий Wi-Fi.

## Возможные доработки (скажи, если нужно)
- Список подключённых устройств с именами.
- История: кто и когда нажимал старт/стоп.
- Виджет на главный экран Android (без открытия приложения).
- iOS-версия (через тот же Kivy + kivy-ios, сборка сложнее — нужен Mac).
